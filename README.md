# PetitOrgRoam

OrgRoam parser, based on [OrgMode](https://github.com/JurajKubelka/OrgMode) by JurajKubelka.

## Loading instructions

### Starting from a Pharo image

Open a playground window (`Ctrl+O+W`) and evaluate:

```smalltalk
Metacello new baseline: 'PetitOrgRoam';
    repository: 'github://capsulecorplab/PetitOrgRoam:main';
    load.
```

Note: Evaluate by highlighting the text, then either right-click on the highlighted text and click `Do it` or press `Ctrl+D`.
